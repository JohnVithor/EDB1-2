## Objetivo
Este programa tem como objetivo compreender o funcionamento da std::list do C++
Implementando alguns métodos surportados pela std::list e testando sua corretude por meio do assert()

### Descrição
Para maiores detalhes sobre esse exercício consulte o pdf "projeto_ED_list.pdf", onde é descrito com detalhes o que será realizado.

## Compilação
Para a compilação utilize o comando 'make' no terminal do Linux.

Para gerar a documentação digite 'make doxy' no terminal.

## Executar o programa
Para executar os testes digite no terminal:
	
    $./bin/List

### Resultados
Após ter iniciado o programa, serão mostradas na tela algumas mensagens sobre os testes realizados, para maiores detalhes de cada teste é necessário verificar o código-fonte no diretório src.
